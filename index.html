C语言基础

进制

2进制B 8进制O 16进制H

以010010010110B 二进制为例，那么8进制和16进制的表示分别为

010/010/010/110B 2226O(八进制表示)

0100/1001/0110B 496H(十六进制表示)

三位二进制表示范围 0-7 四位二进制表示 0-15

数据类型

整数类型

char 存储一个字符 例如‘a’ int 存储一个数字 例如 212

浮点数类型

float 类型在计算机存储中使用单精度IEEE754编码存储 在计算机经常使用double类型存储浮点数

void 无返回值类型

变量

例如 int i, j, k;

声明并定义了变量 i、j 和 k，这指示编译器创建类型为 int 的名为 i、j、k 的变量。

位运算符

条件表达式 ？：

x>y?x:y 表示如果x>y成立则返回x，否则返回y

判断 if

if(表达式) {公式一} else{公式二}

表示表达式成立执行公式一，不成立执行公式二。

while循环

意思是，先计算“表达式”的值，当值为真（非0）时， 执行“语句块”；执行完“语句块”，再次计算表达式的值，如果为真，继续执行“语句块”……这个过程会一直重复，直到表达式的值为假（0），就退出循环，执行 while 后面的代码。

stu 为结构体名，它包含了 5 个成员，分别是 name、num、age、group、score。结构体成员的定义方式与变量和数组的定义方式相同，只是不能初始化。
